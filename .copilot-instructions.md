# Rooted Revival — Copilot Instructions

## Project Overview
Rooted Revival (rootedrevival.us) is a decentralized open-access knowledge archive. It runs on a custom P2P network (GrabNet) with a Cloudflare tunnel to the public web. Internet Archive provides downtime protection.

## Architecture
- **Frontend**: Static HTML pages at repo root (index.html, admin.html, etc.) — no build step, no framework
- **Backend**: Custom Node.js HTTP server in `server/src/` — zero-dependency HTTP framework in `http.js`
- **GrabNet**: Rust P2P engine in `grab/` — content-addressed storage, Merkle-verified bundles
- **Scholar API**: Rust+Actix backend in `scholar/` — academic features, moderation, OpenAPI
- **Desktop**: Electron app in `desktop/`, Tauri app in `desktop-tauri/`
- **Relay**: WebSocket relay in `relay/`
- **Pinning Service**: Node.js in `pinning-service/`

## Key Conventions
- All HTML pages are self-contained single files (CSS + JS inline, no build tools)
- API base URL: `https://scholar.rootedrevival.us/api` (production), `http://localhost:3000/api` (dev)
- Auth: HTTP-only session cookies. U2F elevation via `u2f_verified` cookie (4hr TTL, JS-readable)
- Theme system: CSS custom properties with `data-theme` attribute on `<html>`
- Available themes: terminal (green-on-black), vapor (synthwave), paper (warm white), midnight (blue)
- Font stack: Share Tech Mono (headings/mono), Inter (body)
- All DB: SQLite via better-sqlite3 in `server/data/scholar.db`

## Security Model
- Admin access: password auth (session cookie) OR hardware key (WebAuthn/U2F)
- Write operations (page create/edit/delete, media upload, settings, nav) require U2F elevation
- Read-only admin views (stats, user list, logs) require only session auth + isAdmin
- Flipper Zero acts as U2F key via USB HID — uses `authenticatorSelection.userVerification: 'discouraged'`

## API Patterns
- All JSON responses: `{ success: true, ...data }` or `{ error: "message" }`
- File uploads: multipart/form-data via custom parser in http.js
- Pagination: `?limit=N&offset=N` query params
- UUIDs used for all public-facing resource identifiers

## Don't
- Don't add npm build tooling or bundlers to the frontend
- Don't use external CDN dependencies beyond Google Fonts
- Don't change the HTTP framework — it's intentionally minimal
- Don't modify the GrabNet Rust code without explicit request
- Don't store secrets in code — use environment variables
