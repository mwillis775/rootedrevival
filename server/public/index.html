🛰️ Help host OpenSource Scholar on IPFS

No API keys required. If you run IPFS (Kubo) or a local pinning service, you can help preserve the library by pinning content.

  1. Install IPFS (Kubo) and start a local node
  2. Add/pin the site or specific collections
  3. Share your mirror with the community
ipfs add -r ./public ipfs pin add <CID>
